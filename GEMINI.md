# Anniversary App

This is a multi-purpose repository containing several web-based interactive applications and a Capacitor-based Android wrapper.

## Project Structure
- `public/`: Contains independent web applications:
  - `boids-app/`: A 2D flocking simulation.
  - `emitter-app/`: A 3D emitter application (possibly related to particles or visual effects).
- `android/`: Android project files generated by Capacitor for wrapping the web applications.

## Technologies
- **Web:** Vanilla JavaScript, HTML5, CSS3.
- **Mobile:** Capacitor for native Android integration.

## Building and Running
The project is structured as a collection of independent web apps with a shared Android container.

- **Web Apps:** These are static sites located in `public/`. They can be served using any local HTTP server.
- **Android:** To work with the Android project, ensure you have the Android SDK and Gradle installed. 
  - Navigate to the `android/` directory and use `./gradlew` to build or run the native application.
  - For Capacitor-specific commands, refer to the `capacitor.config.json` at the root.

## Development Conventions
- **Web:** No explicit framework is used; standard browser APIs and external libraries (like `pixi.min.js`) are utilized.
- **Android:** Standard Capacitor/Gradle project structure. Avoid manual modifications to files inside `android/` that are managed by Capacitor.
  - **Rebuilding after Web Changes:** If you make significant changes to `public/index.html` or other web assets, you must synchronize them with the Android project and rebuild:
    ```bash
    npx cap sync android
    ./gradlew assembleDebug
    ```
